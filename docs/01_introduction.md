# Introduction

Welcome to the Donation Management System project. This document provides an overview of the project goals, scope, and high-level context.

The Donation Management System is a modular solution designed primarily for a specific non-profit organization, with extended functionality to support a public demo version. The system is used for donation tracking, donor management, and tax receipt automation, and is configurable for different deployment scenarios.

**Key features include:**

- Modular, environment-driven configuration
- Automated tax receipt generation
- Donor and donation management
- Optional external integrations (e.g., donor sync, email, logging)
- Demo mode for public display

Refer to the rest of the documentation for setup, configuration, and usage details.
